# beauty-border

> beautiful card component

## Quick Start
``` javascript
完整引入
import Vue from 'vue'
import beautyBorder from 'beauty-border'

Vue.use(beautyBorder)

<yl-border-1></yl-border-1>

按需引入
import { YlBorder1 } from 'beauty-border'
```
